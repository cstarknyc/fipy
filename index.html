FiPy for Python 3

Python3.x migration of FiPy (develop), a finite volume PDE solver

View on GitHub Download .zip Download .tar.gz

Please refer to the main FiPy site for the real deal.

This is a fork of the develop branch of FiPy migrated using 2to3 to work with Python 3.x. It has been tested with Python 3.4 on MacOSX.

When running python setup.py install you'll get error reports warning that the absence of the pysparse will break FiPy. This is not true - it works just fine without.

However, make sure you have Gmsh installed before installing FiPy. It is essential.

This Py3 migration of FiPy is maintained by cstarknyc.
